# QuickChat Part 3

This project is a Java console application that includes:

- User registration and login
- Message validation
- Sending, disregarding and storing messages
- Message IDs and message hashes
- Parallel arrays for message data
- Searching by message ID and recipient
- Deleting a message using its hash
- Stored message reports
- JSON file storage and reading
- JUnit automated tests

## How to run

1. Open the project in NetBeans.
2. Run `QuickChatApp.java`.
3. Register a user and log in.
4. Use the main menu to send, view and manage messages.
